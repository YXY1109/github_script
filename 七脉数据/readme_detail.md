## 介绍

### 加密破解爬取七麦数据网应用下载量数据

- B站：https://www.bilibili.com/video/BV1q8411m7xq/
- B站作者是操作的排行榜，我操作的是app的下载量，加密方式是一样的，就是传入的参数不同而已
- QiMaiSpider.py：同步版本
- QiMaiSpider_coroutine.py：协程版本
